# EduHub

EduHub is a full-stack academic platform for students. It is designed to bring learning materials, student progress tracking, academic discussions, and academic bookstore features into one organized web application.

The project is being developed as a realistic student full-stack web technology project using HTML, CSS, JavaScript, Node.js, Express.js, and Supabase PostgreSQL.
